- git checkout <file-name> : to discard unstaged changes in a specific tracked file (the traditional way)[cite: 6].
- git checkout . : to discard unstaged changes in all tracked files at once[cite: 6].

- git restore <file-name> : to restore a specific file to its last committed state (a clearer alternative introduced in Git 2.23)[cite: 6].
- git restore . : to restore all tracked files to their last committed state[cite: 6].

- git clean -dn : to preview which untracked files and directories will be deleted (always do this first!)[cite: 6].
- git clean -df : to force delete all untracked files and directories from your working directory[cite: 6].
